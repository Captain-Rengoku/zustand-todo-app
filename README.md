# zustand todo app

We have build a full CRUD (Create, Read, Update, Delete) Todo App using Next.js with modern Zustand state management in this beginner-friendly tutorial.

Set up a Next.js project from scratch
Install and configure Zustand for global state management

Later on we build a full-stack ToDo app using Next.js, Prisma, PostgreSQL, and Zustand for state management.

## We learned

- Build a CRUD Todo App step-by-step:
  - Create new tasks
  - Read and display tasks
  - Update existing tasks
  - Delete tasks easily
  - Manage state across multiple components
  - Keep your code clean, simple, and easy to maintain
- how to set up a PostgreSQL database with Prisma,
- create full CRUD routes in Next.js,
- and integrate everything into a global state using Zustand.
